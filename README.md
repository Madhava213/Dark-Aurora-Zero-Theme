<h2 style="color: #028090">Installation :</h2>

1. Open the extension bar on VS Code
2. Search for **_Dark Night Zero Theme_**
3. Click _Install_
4. Reload your editor

 <h2 style="color: #028090">Activate Theme :</h2>

Launch Quick Open:
